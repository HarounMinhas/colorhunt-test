# Bootstrap 5.3.3 Showcase met Color Palette Widget

Een complete demonstratie van alle Bootstrap 5.3.3 componenten met een dynamische kleurenschema selector.

## Functies

- **Floating Color Widget**: Een zwevende widget rechtsboven die openklapt zonder andere elementen te verstoren
- **Dynamische Kleurenschema's**: 6 voorgedefinieerde kleurenschema's die met één klik de hele pagina aanpassen
- **Volledige Bootstrap Showcase**: Alle belangrijke Bootstrap componenten getoond:
  - Typography (headings, paragraphs, blockquotes)
  - Buttons & Badges
  - Cards
  - Alerts
  - Forms (inputs, selects, checkboxes, switches, range sliders)
  - List Groups
  - Progress Bars & Spinners
  - Tables
  - Accordion
  - Breadcrumb & Pagination
  - Modals
  - Navigation

## Kleurenschema's

1. **Standaard** (Default): #5A9CB5, #FACE68, #FAAC68, #FA6868
2. **Dark Blue**: #222831, #393E46, #00ADB5, #EEEEEE
3. **Warm Sunset**: #F7B801, #F18701, #F35B04, #C1121F
4. **Ocean**: #06283D, #1363DF, #47B5FF, #DFF6FF
5. **Berry**: #2B3467, #BAD7E9, #FCBAAD, #EB455F
6. **Deep Sea**: #1B262C, #0F4C75, #3282B8, #BBE1FA

## Gebruik

1. Open `index.html` in je browser
2. Klik op het palette-icoon rechtsboven
3. Selecteer een kleurenschema uit de lijst
4. De kleuren worden naadloos toegepast op de hele pagina

## Technologie

- Bootstrap 5.3.3
- Bootstrap Icons 1.11.3
- Vanilla JavaScript (geen dependencies)
- CSS Custom Properties voor dynamische theming

## Aanpassen

Om je eigen kleurenschema's toe te voegen, voeg een nieuwe `.palette-item` toe in `index.html` met je gewenste kleuren in het `data-colors` attribuut.
